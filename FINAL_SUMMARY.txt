🎯 ЧТО РАБОТАЕТ СЕЙЧАС:

✅ Уровень 1: Pre-commit Hook (локально)
   git commit → Тесты → Коммит создаётся/отменяется
   
✅ Уровень 2: Deploy Script (перед деплоем)
   ./deploy.sh → Тесты → Деплой/отмена

🚫 GitHub Actions УДАЛЁН (не нужен для solo-разработчика)

═══════════════════════════════════════════════════════════════════

🚀 БЫСТРЫЙ СТАРТ:

1. Установить pre-commit hook:
   ./scripts/install-hooks.sh

2. Попробовать коммит (тесты запустятся):
   echo "# test" >> README.md
   git add README.md
   git commit -m "test: check hook"
   git restore README.md

3. Попробовать деплой:
   ./deploy.sh "test message"
